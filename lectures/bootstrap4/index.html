Bootstrap grid

Overview

Bootstrap grid rules I

  • Rows must be placed within a container.
  • There are two types of containers .container (fixed-width) or .container-fluid (full-width) for proper alignment and padding.
  • Use rows .row to create horizontal groups of columns.
  • Content should be placed within columns, and only columns may be immediate children of rows.
  • From Bootstrap 4 there is allowed to create columns without specified size .col. Row will be divided equally into those columns.

Bootstrap grid rules II

  • Grid columns with specific width can be created by specifying the number of twelve available columns you wish to span .col-1 - .col-12. For example, three equal columns would use three .col-4.
  • If more than 12 columns are placed within a single row, each group of extra columns will, as one unit, wrap onto a new line.

Bootstrap grid rules III

  • Using grid can be created different layouts on different viewports sizes with the same code.
  • There are 5 viewport sizes defined in bootstrap:
    • Extra small devices (portrait phones, less than 576px) - no media query col-*
    • Small devices (landscape phones, 576px and up) - media query col-sm-*
    • Medium devices (tablets, 768px and up) - media query col-md-*
    • Large devices (desktops, 992px and up) - media query col-lg-*
    • Extra large devices (large desktops, 1200px and up) - media query col-xl-*
  • Every grid column can have assigned all of the viewports.
  • Grid classes apply to devices with screen widths greater than or equal to the breakpoint sizes, and override grid classes targeted at smaller devices. Therefore, e.g. applying any .col-md-* class to an element will not only affect its styling on medium devices but also on large devices if a .col-lg-* class is not present.

Bootstrap components

tooltips & popovers

Tooltips are small helper titles shown when mouse hovers over element.

Popovers works the same way as titles, but they are bigger and can carry more data.

For performance reasons you have to initialize tooltips & popovers manually:

<script>
$(function () {
  $('[data-toggle="tooltip"]').tooltip();
  $('[data-toggle="popover"]').popover();
})
</script>
